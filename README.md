# OpenInput [![NuGet Version](http://img.shields.io/nuget/v/OpenInput.svg)](https://www.nuget.org/packages/OpenInput)
A portable way of accessing HID devices

> This project is still in development!

## Supported Platforms

* Windows 
    * DirectInput
    * XInput
    * RawInput
* OpenTK

### Supported Devices

* [Mouse](#Mouse)
* Keyboard
* TouchDevice
* GamePad
